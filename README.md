Forge to Steel is an importer for the Foundryvtt Draw Steel system.  It's being designed to accept a ds-hero file from the Forged Steel character builder and import it.
[ForgeToSteelInstructions.webm](https://github.com/user-attachments/assets/abd4a6d1-7a3c-4afa-a0f4-ba751db62fd2)

This does require a manual install, download the files and put them into a forge-to-steel folder in the modules folder.  Then, activate the module in your world.
